# デザイン・オーバーレイ・色調ルール

## 画像サイズ
- **1080 x 1920px**（TikTok縦型 9:16）

## TikTok UIセーフゾーン（最重要）

TikTokのUI要素（いいね/コメント/キャプション等）が画面端に重なるため、テキストはセーフゾーン内に配置する。

```
1080×1920 キャンバス上のセーフゾーン:
  Top:    130px (For You/Followingタブ)
  Bottom: 400px (キャプション/ユーザー名/サウンド)
  Right:  150px (いいね/コメント/シェアボタン列)
  Left:    60px (エッジマージン)

安全な描画領域:
  X: 60  〜 930   (幅 870px)
  Y: 130 〜 1520  (高さ 1390px)
```

```
┌──────────────────────────────────┐ ← 0px
│ [For You] [Following]            │ ← 危険ゾーン 0〜130px
├──────────────────────────────────┤
│                                  │
│    ★ SAFE ZONE ★               │  [♡] いいね
│    テキストはここに配置           │  [💬] コメント
│    X: 60〜930                    │  [↗️] シェア
│    Y: 130〜1520                  │  [🔖] 保存
│                                  │
├──────────────────────────────────┤ ← 1520px
│ @ユーザー名                       │
│ キャプション                      │ ← 危険ゾーン 1520〜1920px
│ ♪ サウンド名                     │
└──────────────────────────────────┘
```

## ブランドカラー

全スライドで統一するカラーパレット:

| 色 | コード | 用途 |
|----|--------|------|
| ティール | `#14B8A6` | アクセント/Q文/強調テキスト/背景画像のライト |
| 白 | `white` / `#F5F5F5` | メインテキスト |
| ダーク | `#1A1A1A` | 背景基調 |

- 全スライド共通でティール(`#14B8A6`)をアクセントカラーに使う
- 背景画像もティール色のライティングを指定して統一感を出す

## 背景画像

- ChatGPT(DALL-E)で生成
- テーマに合ったフォトリアル/3Dレンダリング風
- 人物はシルエットや後ろ姿のみ（顔なし）
- テキストは画像内に入れない（後からImageMagickで合成）
- **全スライドで色調を統一する**: 暗めの背景 + ティール色のアクセントライト

## オーバーレイルール（重要）

| 背景タイプ | 処理 | ImageMagickコマンド |
|-----------|------|-------------------|
| ChatGPT生成画像（デフォルト） | 60%で暗くする | `-fill black -colorize 60%` |
| 明るい背景 | 70%で暗くする | `-fill black -colorize 70%` |
| 非常に暗い背景（テキストなしで十分暗い） | オーバーレイなし | なし |

- **ChatGPT(DALL-E)生成の画像は「暗い」と指定しても背景が明るいことが多い**
- **デフォルトで `-fill black -colorize 60%` を全スライドに適用する**
- テキストの可読性が最優先 迷ったら暗くする
- `-colorize` の値で暗さを調整（50%=少し暗く 60%=標準 70%=しっかり暗く）
