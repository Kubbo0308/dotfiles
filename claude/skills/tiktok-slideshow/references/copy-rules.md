# タイトル・説明文・ハッシュタグルール

## タイトル

- **短く刺さる1文**: 検索にも引っかかるキーワードを含める
- **「〜な人へ」「〜って人へ」型**: ターゲットを明示して自分事化させる
- **結論は書かない**: 気になってスワイプさせる

## 説明文

### 役割（最重要）

説明文は**訴求の場**。スライドの内容を要約・繰り返す場ではない。

**目的**: 視聴者にどうなってほしいか、どう感じてほしいかを伝える。
- スライドは「気づき」を与える
- 説明文は「行動・感情」を促す

### NG / OK

| NG（スライドの繰り返し） | OK（訴求） |
|------|------|
| 「疲れてる」→ わかる でも軽く動いた方が... | この投稿が、明日のあなたの一歩のきっかけになったら嬉しい |
| 体重はほぼ変わらなかった でも見た目は... | 数字に振り回される必要なんてない あなたの努力はちゃんと体に刻まれてる |
| 1ヶ月じゃ体は変わらない でも行動は... | 今の小さな積み重ねが 3ヶ月後のあなたを作るから 焦らなくていいよ |

### 構成テンプレート

```
{呼びかけ — 読者の状況や感情に寄り添う1〜2行}

{気持ちの共有 — 自分も同じだった、理解してるよというメッセージ}

{訴求 — 視聴者にどうなってほしいか / どう感じてほしいか}

{背中を押す一言 — 行動や心の変化を促す}

参考になったら保存してね
コメントで{何か}教えてくれたら嬉しい
```

### ルール

- **スライドの内容を繰り返さない**: スライドで言った言葉を説明文に再掲しない 別の角度から訴求する
- **「あなた」を主語に**: 一般論ではなく、見ている人個人へのメッセージにする
- **感情に訴える**: 情報ではなく感情・気持ちを動かす言葉を選ぶ
- **長めに書く**: 長い説明文の方が平均視聴者数が3倍増える（200〜400字目安）
- **口調は丁寧なタメ口**: 「〜だよね」「〜かも」「〜だと思う」
- **煽り口調は絶対使わない**: 「〜できる?」「〜でしょ?」のような挑発的な問いかけNG
- **上から目線にならない**: 「〜すべき」→ 「〜できたら嬉しい」
- **断定しない**: 「これが正解」→ 「自分はそう感じてる」

### 訴求パターン

投稿の内容に応じて使い分ける:

- **背中を押す型**: 「一歩踏み出す勇気をあげたい」
- **安心させる型**: 「焦らなくていい あなたのペースで大丈夫」
- **承認型**: 「あなたの努力はちゃんと意味がある」
- **未来提示型**: 「3ヶ月後のあなたが、今のあなたに感謝するはず」
- **共感型**: 「自分もずっとそうだった だから気持ちわかる」
- **問いかけ型**: 「あなたは今、どう感じてる？コメントで教えて」

## ハッシュタグ

### 構成

```
#{メインキーワード} #{メインキーワード}初心者 #{メインキーワード}あるある
#{関連ワード1} #{関連ワード2} ...
#筋トレ #筋トレ初心者 #ジム初心者 #ワークアウト #トレーニング #フィットネス
```

- メインキーワード（テーマ固有）を先頭に
- 汎用タグ（筋トレ/ジム初心者/ワークアウト等）は後半に
- 10〜12個が目安

### 固定フッター

説明文の最後に必ず入れる:
```
参考になったら保存してね
他にも気になることあったらコメントで教えてね
```
