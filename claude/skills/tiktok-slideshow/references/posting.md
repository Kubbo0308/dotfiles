# ハッシュタグ・投稿ルール・チェックリスト

## ハッシュタグテンプレート

```
#{メインテーマ} #{メインテーマ}初心者 #{メインテーマ}あるある
#習慣化 #{関連ワード1} #{関連ワード2}
```

### 筋トレ例
```
#筋トレ #筋トレ初心者 #筋トレ習慣 #三日坊主 #筋トレあるある
#筋トレ続かない #習慣化 #筋トレ記録 #ワークアウト #トレーニング
```

## 投稿ルール

- **形式**: TikTokアプリの写真スライドショー（Web版TikTok Studioでは写真投稿不可）
- **投稿タイミング**: 平日 19:00〜21:00 推奨 / 次点 土日 10:00〜12:00
- **BGM**: TikTokのAI自動選曲をそのまま使うか Lo-fi / Chill Hip-hop系
- **配信方法**: AirDrop で Mac → iPhone に画像転送 → TikTokアプリで投稿

## 投稿後アクション

- **最初の2時間以内のコメントには全返信**: エンゲージメントブースト
- **保存率を重視**: 2026年時点で保存はいいねの3倍の重み付け
- **スライド別離脱データを確認**: Slide 1→2の遷移率が最重要指標

## チェックリスト

### コンテンツ
- [ ] 5枚構成になっているか（フック+Q&A×3+まとめ）
- [ ] Q&Aが具体的で深い回答になっているか（浅いNG）
- [ ] まとめスライドの全項目に説明文が入っているか
- [ ] 句読点が入っていないか
- [ ] 強い言葉を使っていないか（〜しろ/〜やれ → 〜してみて/〜するといいよ）
- [ ] 宣伝臭がないか（価値提供が最優先）

### デザイン
- [ ] 全スライドで色調が統一されているか（ダーク+ティール）
- [ ] テキストがセーフゾーン内に収まっているか（右150px 下400px 空ける）
- [ ] フォントサイズが十分大きいか（説明文 54pt以上）
- [ ] 強調ワードがティール色+大きいフォントで目立っているか
- [ ] 暗い背景にオーバーレイしていないか
- [ ] 明るい背景が適切に暗くなっているか（colorize 50-70%）
- [ ] テキストにstroke(縁取り)がついていないか

### 投稿準備
- [ ] タイトルが用意されているか
- [ ] 説明文が魅力的か（悩みの列挙 + 共感 + 行動促進）
- [ ] ハッシュタグが準備されているか
- [ ] スワイプ方向が「→」になっているか
